# Emoji-Canvas
html5 canvas練習

以ios中的表情符號為練習對象，繪製：
1. 喜(Smiling Face)
2. 怒(Pouting Face)
3. 哀(Loudly Crying Face)
4. 樂(Grinning Face with Smiling Eyes)
共四個表情圖案

表情符號參考來源：https://emojipedia.org/apple/
